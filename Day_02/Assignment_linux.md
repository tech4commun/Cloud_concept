# Linux Commands Practical Exercises

**1. Create a new folder named `projects`**
```
mkdir projects
```

**2. Move `report.txt` into subfolder `documents`**
```
mv report.txt documents/
```

**3. Copy `photo.jpg` from home directory to `/media/backup/`**
```
cp ~/photo.jpg /media/backup/
```

**4. Permanently delete a file named `old_notes.txt`**
```
rm old_notes.txt
```

**5. Safely remove an empty folder named `temp_data`**
```
rmdir temp_data
```
Extra flag if folder contains files:
```
rm -r temp_data
```
