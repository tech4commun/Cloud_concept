Ubuntu · APT Package Manager

Package installation commands

All essential commands for managing software on Ubuntu using apt — the default package manager.

1 · Update & upgrade
Update package list
Fetches latest info from servers — always run this first
sudo apt update
Note: This does not install anything — it only refreshes the list of available packages.
Upgrade installed packages
Installs newer versions of all installed packages
sudo apt upgrade
Tip: Combine both — sudo apt update && sudo apt upgrade
Full upgrade
Upgrades + handles dependency changes
sudo apt full-upgrade
Use when: Regular upgrade skips a package due to dependency conflicts.
2 · Install packages
Install a package
Downloads and installs the specified package
sudo apt install <package-name>
Example: sudo apt install nginx
Install multiple
Space-separated list
sudo apt install git curl wget
Skip confirmation
Auto-yes to all prompts
sudo apt install -y nginx

On AWS EC2 (Amazon Linux): Use yum instead — e.g. sudo yum install nginx. Ubuntu-based EC2s still use apt.

3 · Remove packages
Remove
Removes package, keeps config files
sudo apt remove <pkg>
Purge
Removes package + all config files
sudo apt purge <pkg>
Autoremove
Removes unused dependencies left behind
sudo apt autoremove
Best practice: Run this after removing packages to keep the system clean.
4 · Search & info
Search package
Find by keyword
apt search <keyword>
Show info
Version, size, description
apt show <package>
List installed
All installed packages
apt list --installed
List upgradable
Packages with updates available
apt list --upgradable
5 · Clean up cache
Clean cache
Deletes all cached .deb files
sudo apt clean
Autoclean
Removes only outdated cached files
sudo apt autoclean
